**DQN: CartPole and MsPacman**

- Deep Q-Network implementations for CartPole and MsPacman with training, evaluation, and plotting.

**Environment Setup**

- Python 3.9+ recommended.
- Create a virtual environment and install dependencies:
  - `python -m venv .venv`
  - `source .venv/bin/activate`  (Windows: `.venv\Scripts\activate`)
  - `pip install --upgrade pip`
  - `pip install -r requirements.txt`
- For MsPacman (Atari), ensure ROMs/envs are available (easiest):
  - `pip install "gymnasium[atari,accept-rom-license]"`

**MsPacman: Train + Evaluate**

- Train (example): `python train_mspacman_dqn.py --mode train --train-episodes 2000 --outdir runs/mspacman_dqn`
